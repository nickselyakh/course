\noindent ОПТИМАЛЬНОЕ УПРАВЛЕНИЕ, УПРАВЛЕНИЕ ПО ПРОГНОЗИРУЮЩЕЙ МОДЕЛИ, АЛГОРИТМ, МНОГОПАРАМЕТРИЧЕСКИЙ ИНСТРУМЕНТАРИЙ, МНОГОПАРАМЕТРИЧЕСКОЕ ПРОГРАММИРОВАНИЕ

\bigskip

Объект исследования --- классическая задача синтеза.

\bigskip

Цель работы --- исследование основныx положений явного МРС, идеи построения обратных связей на основе параметрического программирования с дальнейшим развитие на проблему синтеза оптимальной системы.

\bigskip

Методы решения --- МРС в сочетании с многопараметрическим линейным и квадратичным программированием. Исользование MPT для решения задач оптимального управления.

\bigskip

Основные результаты работы --- исследованы основныx положений явного МРС, идеи построения обратных связей на основе параметрического программирования с дальнейшим развитие на проблему синтеза оптимальной системы. Предлоржены метод решения задачи оптимального управления, реализованы решения рассматриваемых задач при помощи MPT.

\centerline{\large{\textbf{на англ. яз.}}}
\bigskip
\bigskip

\textbf{\textit{Key words}} --- OPTIMIZATION PROBLEMS, MODEL PREDICTIVE CONTROL, ALGORITHM, MULTI-PARAMETRIC TOOLBOX, MULTI-PARAMETRIC PROGRAMMING

\bigskip

\textbf{\textit{Object of research}} --- the classic problem of synthesis.

\bigskip

\textbf{\textit{Aim of the work}} --- explicit MPC, the idea of building feedback based on parametric programming with further optimal control problems.

\bigskip

\textbf{\textit{Solving methods}} --- MPC in combination with multiparametric linear and quadratic programming. The usage of MPT for resolving optimal control problems.

\bigskip

\textbf{\textit{Results of rhe work}} --- conducted the main provisions of explicit MPC research, the idea of building feedbacks based on parametric programming with further optimal control problems. Proposed methods for solving the optimal control problem and solutions to these problems which implemented using MPT.
